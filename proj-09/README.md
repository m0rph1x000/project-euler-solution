# Problem 9 - Special Pythagorean triplet

## Description

A _Pythagorean triplet_ is a set of three natural numbers, _a < b < c_, for which,

$$ a^2 + b^2 = c^2 $$

For example,

$$ 3^2 + 4^2 = 9 + 16 = 25 = 5^2 $$

There exists exactly one Pythagorean triplet for which _a + b + c_ = 1000.
Find the product abc.
